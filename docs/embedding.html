Polarizable embedding¶

An SCF calculation with a polarizable environment is performed in VeloxChem with an input file of the form

@jobs
task: scf
@end

@method settings
basis: aug-cc-pvdz
potfile: pe.pot
@end

@molecule
charge: 0
multiplicity: 1
xyz:
...
@end
